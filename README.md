# NexusChat

## How to run?

1.  create a virtual environment

```bash
conda create -n llmapp python=3.12 -y

```

2. Activate the environment

 ```bash
conda activate llmapp

```

3. Install the required packages

```bash
pip install -r requirements.txt

```
## chainlit commands

1. initialise chainlit

```bash
chainlit init

```
2. Run NexusChat

```bash
chainlit run app.py

```
